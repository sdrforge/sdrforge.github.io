# ADS-B Aircraft Tracker

Estimated Time: ~5 minutes

## Summary

The ADS-B Aircraft Tracker lab demonstrates how aircraft broadcast telemetry information and how SDR receivers decode and visualize this data.

This lab provides a safe environment for understanding aviation surveillance systems and aircraft tracking workflows.

---


## Launching the Lab

Start SDRForge:

```bash
python3 SDRForge.py
```

Select:

```txt
ADS-B Aircraft Tracker
```

![ADS-B Menu](/img/adsb-menu.png)

---

## Interface Overview

The ADS-B dashboard contains:

- Interactive map - you can follow planes in real time
- Flight information panel - this is the planes data like altitude, call sign, and direction
- Live tracking display

![Aircraft Map](/img/adsb-map.png)

---

## Understanding ADS-B

ADS-B stands for:

```txt
Automatic Dependent Surveillance-Broadcast
```

Aircraft periodically transmit:

- Position
- Altitude
- Ground Speed
- Heading
- Callsign

We could use this info to spy on aircraft however this data is usally automatically transmitted and public

## Troubleshooting

### No aircraft visible

Verify the simulation has started and refresh the browser.

### Map not updating

Restart the lab and reload the page.

### Aircraft was present and no longer appears


Shut down the lab and give it a few minitues the api has a request limit
